# Sreeraj S — Portfolio

This is a simple single-page portfolio for Sreeraj S (Data Science & ML Enthusiast).

## How to use
- Edit `index.html` to update text, links or email.
- Replace project placeholders with your repositories or report files.

## Publish with GitHub Pages
1. Create a repository named `username.github.io` (replace `username`).
2. Add these files and push to main branch.
3. Your site will be available at `https://username.github.io`.
